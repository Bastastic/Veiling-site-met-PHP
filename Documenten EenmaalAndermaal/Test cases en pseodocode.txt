Testcase Blokeer Gebruiker:
- gebruiker via blokkeer butten blokeren
	Input muisklik	
		Post naar Database zoals verwacht
- Gebruiker blokkeren via naam
	Input naam en reden
		Post naar Database zoals verwacht
- Geen bestaande gebruiker blokeren via naam
	Input neppenaam en reden
		geeft errorcode zoals verwacht
- Geen reden proberen mee te geven
	input naam
		geeft errorcode zoals verwacht
- Geen naam proberen mee te geven
	input reden
		geeft errorcode zoals verwacht
-Tijdsduur proberen mee te geven
	input naam reden tijdsduur
		Post naar database zoals verwacht
-Extreem hoge tijdsduur proberen mee te geven
	input naam reden tijdsduur(9999)
		Post naar database zoals verwacht
- Vul min 1 in bij duur
	input naam reden tijdsduur(-1)
		Geeft errorcode zoals verwacht
-Deblokeren via button
	Input muiskik
		post naar database zoals verwacht
-Deblokeren via naam
	Input naam
		Post naar database zoals verwacht
-Deblokeren via nepnaam
	Input niet geblokeerde naam
		geeft errorcode zoals verwacht

Pseodocode

-------------------------
blokeren.php

<form handmatig(method POST)>
 inputfile(POST = $username)
 inputfile(POST = $reden)
 inputfile(POST = $duur)
</form>

<form Blokeerbutton(method POST)>
 button(value = $username)
 inputfile(POST = $reden)
 inputfile(POST = $duur)
</form>

if(!$duur){
 $duur = NULL
}

 $sql -> $username, $reden, $duur into Geblokeerde_gebruikers


-------------------------------
deblokeren.php

<form handmatig(method POST)>
 inputfile(POST = $username)
</form>

<form Deblokeerbutton(method POST)>
 button(value = $username)
</form>

$sql -> Delte WHERE username = $username from Geblokeerde_gebruikers
